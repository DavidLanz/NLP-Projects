# Introduction
Deep semantic match has broad applications in **information retrieval**, **question answering**, **dialog system**, **paraphrase**, etc.

# Statistical models
- TFIDF
- BM25
- Query likelihood
- Jaccard distance

# Deep learning models
- **Interaction-based** is **perfered**
    - ARC-II
    - DeepMatch
    - MatchPyramid
    
- **Representation-based**
    - DSSM
    - CDSSM
    - ARC-I
